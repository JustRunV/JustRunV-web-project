Project 2 - HTML Page

Webpage Theme

My webpage is themed around my running journey.  It highlights why running is meaningful to me,
my favorite places to run, and the trails that inspire me.  The page includes images, a navigation bar, a list, and a table describing different trail options.  

What I Learned

While creating this page, I learned how to structure an HTML document correctly using elements like <html>, <head>, and <body>. I praticed adding heading, paragraphs, lists, images, links and tables.  I also learned how file paths work, especially when linking between pages in different folders.  I gained confidence using inline styling to control background colors, spacing, and text appearance. 

Challenges I Faced

One of the biggest challenges was getting the navigation links to work correctly. Small mistakes in file paths or folder structure caused "file cannot be accessed" errors, and I had to troubleshoot missing quotes, misplaced tags, and incorrect paths.  I also had to fix my page structure when content accidentally ended up outside the <body> tag. These challenges helped me understand how precise HTML needs to be and how important it is to keep files organized.
